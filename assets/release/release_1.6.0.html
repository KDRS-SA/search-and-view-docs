1.6.0 - December 2024
New

  • external files (files outside SIARD referenced by path)

  • language support. norwegian, sweedish, english

  • render 'my_view' can render views relative to template
  • upgrade solr, dbptk
  • autostart app on reboot
  • allow archives with same id, to explore multiple templates

  • download could get the wrong document, if search on a renamed field
  • dbptk: fixed problem with indexing some blob types
  • install: will skip docker update if no Internet
  • filter on 'filnavn' did not work if field 'navn' present

  • æøå did not display when opening file in browser

  • cache hit will now require a match in solr id

Views

  • extens fix: new view for class list history
  • extens: listing final classes on top
  • extens: converting utc dates to local time zone

Nytt
  • Oppdatering av biblioteker
  • Oppdatering av dbptk
  • Kolonner med mellomrom blir nå trimmet før join. F.eks "ADB " kan da kobles mot "ADB"

      • xml tag foreignKey heter nå foreignkey
      • xml tag showSearch heter nå showsearch
      • xml tag customView heter nå customview

  • Alle maler er oppdatert til å følge ny standard
  • Extens klasseliste - Tabell 'elevhist' og 'historia' slås sammen for å finne flere klasselister

  • Søk skal nå bedre ta med seg kontekst som bruker står i
  • Tilbakeknapp skal nå virke også etter et søk

  • Konfigurasjon med store bokstaver, eller en miks, vil ikke lenger bli støttet i xml eller visningsmaler
    • Visual Studio Code kan gjøre endringer automatisk. Søk på regexp
